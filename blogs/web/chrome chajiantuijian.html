chrome （谷歌）插件推荐

MuYan2020-12-30web

工具类插件

Tampermonkey

Tampermonkey（油猴）是一个脚本管理器，它里面可以存放很多脚本文件，而这些脚本才是真正的主角。它能做什么取决于你的脚本是干什么的，都有些什么脚本？例如视频播放、音乐下载、文库资料、小说阅读、网盘下载、资源搜索等等。

Nimbus

截幕 & 屏幕录像机，全屏幕捕获网页或任何部分。编辑的截图。记录的视频记录的视频从您的屏幕。

wappalyzer

Identify web technologies(识别网站所用技术)。

Dark Reader

适用于任何网站的黑暗主题。关爱眼睛，就使用 Dark Reader 进行日常浏览。

FeHelper(前端助手)

JSON 自动格式化、手动格式化，支持排序、解码、下载等，更多功能可在配置页按需安装！

搜图助手

支持多引擎以图搜图；支持淘宝，1688，alibaba 电商网站以图搜商品；支持一键提取网页所有图片并可以任意选择下载，方便，简单，快捷。

Infinity 新标签页 (Pro)

Infinity New Tab Pro 让你可以自定义你的新标签页

OneTab

节省高达 95％的内存，并减轻标签页混乱现象

二维码生成器 (Quick QR)

Chrome 上好评率最高的二维码生成器：可以方便地把当前页面转化成二维码，也可以把网页上任何文本或链接，甚至是您输入的任意内容都转化成二维码

划词翻译

简单好用的翻译扩展程序。

CocoCut

适用在 Chrome、Edge 浏览器，可用来存储线上视频、音乐资源下载 HLS / M3U8 影片，打开网页时会自动探索可以下载的影片和音乐，只需点击图示就能开始下载。

  • 建议搭配M3U8 批量下载器食用

Vue.js devtools

Vue 扩展调试工具

React Developer Tools

React 扩展调试工具

扩展管理器

用于管理谷歌插件

uBlock Origin

去除大部分视频网站的广告

Link to Text Fragment

允许链接到页面上的任意文本。此扩展通过右键快捷创建指向页面上当前所选文本的特殊链接。 打开此类特殊链接时，兼容的浏览器会将所选文本滚动到视图中并突出显示。

Saladict

沙拉查词-聚合词典划词翻译

OneNote Web Clipper

可将网络上的任何内容保存到 OneNote。将其剪贴到 OneNote 中、进行整理和编辑，然后通过任何设置进行访问。

Enhanced GitHub

在 GitHub 网站上提供有用功能的扩展，可以显示仓库大小、每个文件的大小、下载链接和复制文件内容的选项。

Sourcegraph

github 仓库内快捷查询等功能

科学上网

GitHub 加速

国内 Github 下载很慢，用上了这个插件后，下载速度嗖嗖嗖的~！可加速 GitHub 访问\下载

iGG 谷歌访问助手

永久免费提供谷歌搜索，gmail，Chrome Store,Android/Golang 等谷歌产品加速访问.

RUSVPN

免费的科学上网

SetupVPN

免费的科学上网

VPN Proxy VeePN

免费的科学上网

其他插件

喵喵折

“喵喵折”是简单易用的购物助手，为您提供商品的历史价格、同款比价、商品收藏、降价提醒以及一键海淘等服务。
